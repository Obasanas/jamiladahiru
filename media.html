📲 Instagram API Integration Guide

To enable automatic Instagram post syncing on this page, follow these steps:

1. Convert your Instagram account to a Business or Creator account.

2. Create a Meta Developer App at developers.facebook.com and add the Instagram Graph API product.

3. Generate a Long-Lived Access Token and store it as an environment variable INSTAGRAM_TOKEN.

4. Fetch latest posts via:

GET https://graph.instagram.com/me/media?fields=id,caption,media_type,media_url,permalink&access_token=YOUR_TOKEN

5. Use a serverless function (Vercel/Netlify) to cache and serve the results, then replace the placeholder grid above.

See the README.md in the project repository for the full Next.js implementation with Instagram Graph API integration code.
