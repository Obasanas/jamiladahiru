📲 Instagram API Integration

To enable automatic Instagram post syncing, configure the Instagram Graph API:

1. Convert your account to Business/Creator on Instagram.

2. Create a Meta Developer App at developers.facebook.com.

3. Generate a Long-Lived Access Token → store as INSTAGRAM_TOKEN.

4. Fetch posts via:

GET https://graph.instagram.com/me/media?fields=id,caption,media_type,media_url,permalink&access_token=YOUR_TOKEN

See the README.md for the complete Next.js implementation.
